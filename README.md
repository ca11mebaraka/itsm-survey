# ITSM Survey Tool

Интерактивный инструмент для проведения обследования ITSM-процессов на базе опросника ГОСТ Р 57329 (ИСО/МЭК 20000).

## 🌐 Онлайн-версия (GitHub Pages)

**[→ Открыть инструмент](https://ca11mebaraka.github.io/itsm-survey/)**

Работает прямо в браузере — установка не требуется.
Все данные хранятся в файле `.db` на вашем компьютере.

## ⚡ Быстрый старт

### Онлайн (GitHub Pages)
1. Откройте https://ca11mebaraka.github.io/itsm-survey/
2. Нажмите **«Начать новое обследование»**
3. Введите название сессии
4. Отвечайте на вопросы
5. Регулярно нажимайте **«💾 Сохранить файл»** — файл `.db` скачается на компьютер
6. В следующий раз: **«Продолжить (загрузить файл .db)»**

### Локально (Python)
```bash
pip install flask
python main.py
# Откроется браузер на http://localhost:5000
```

## 💾 Формат данных

База данных SQLite (`.db` файл) содержит:
- `sessions` — сессии обследования
- `answers` — ответы с временными метками
- `app_state` — состояние интерфейса

Совместима с `pandas`, `DB Browser for SQLite`, `Power BI`.

```python
import sqlite3, pandas as pd
conn = sqlite3.connect("itsm_survey_МоёОбследование.db")
df = pd.read_sql("SELECT * FROM answers", conn)
```
